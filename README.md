 🎓 Student Management System

 🌐 Live Demo
https://anant-sms.infinityfreeapp.com/

 📌 Features
- Secure Login System
- Add / Update / Delete Students
- AJAX Search
- Bootstrap Responsive UI
- Role-Based Access

 🛠 Tech Stack
- PHP
- MySQL
- Bootstrap 5
- JavaScript (AJAX)

 ⚙️ Setup
1. Clone repository
2. Import database
3. Update config/db.php
4. Run locally using XAMPP
